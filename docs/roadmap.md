# Roadmap

## M0 — Foundation

- Monorepo, local services, domain contract, database schema
- Deterministic metric primitives and test fixtures
- Provider-independent import contract

## M1 — One complete sample

- Choose one legally accessible provider
- Import a single pre-kickoff score and 1X2 snapshot
- Persist final result and calculate entropy, concentration, rank, top-k, Brier, and log loss
- Render one match report with provenance

## M2 — Historical evaluation

- Scheduled collection and idempotent retries
- Calibration buckets and rolling evaluation
- Dataset/version manifests
- Daily and tournament reports

## M3 — Research workbench

- Custom grouping and filtering
- Regression, bootstrap, and Bayesian experiment jobs
- Pattern review workflow (candidate, validated, rejected)
- Exportable charts and research summaries

## Explicitly deferred

- Broad provider coverage before one adapter is reliable
- Real-time trading features
- Betting recommendations, staking, or strategy optimization
